This repository contains the project **“Linguistic Advisor” (المستشار اللغوي)** developed by **Team Ranan** as part of the Samsung – Misk – Tuwaiq program.

An AI-powered Arabic linguistic advisor that helps users improve grammar, wording, and formal writing quality.
